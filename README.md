 # Pongamingas 
 
 ## Objective
 Create a multiplayer minesweeper game. 
 
 ## Used technologies
 - React
 - Node
 - Mongo
 - Meteor
 
 ## How to run 
 Insied of ./pongaminas/, run:
 
 ```
 meteor install
 meteor run
 ```
